現代資訊學院

藉由選擇你的方向，設定你有興趣的學習內容，學到教室裡學不到的資訊能力，打造在不確定的未來你所需要的自學能力以及解決問題的能力。

自學能力

不管新創或是舊產業甚至政治到學界都在快速變化， 程式碼規格一直在改，但是你對資訊背後所有原理的認知可以持續累積。

只要你知道怎麼選對戰場，解讀每個資訊，善加利用吸收成自己的一部份，就是自學。

編程開發

你不一定要撰寫程式碼，但一定要了解 開發者的程式邏輯 。

商業計畫思考

你也不是一定要創業，但一定要擁有創業家的計畫思考。

專案執行

在軟體工程師的世界裡，專案執行下去才是真的，有實作能力最重要，什麼科系背景畢業不重要。

這裡是不在教室的未來大學，現代資訊學院，現代資訊化學習的線上學院，所有你需要的資源都已上線，可供你遠端連線學習。

course

為培養現在市場急缺的網站開發人才
線上開課，現代網站開發課程

現代網站開發課程

預計 2 月 21 日開始，為期半年到一年的線上課程，
以及進修線下活動。
學員可以自由選擇要如何參與課程及調整進度。

程式編寫

程式邏輯設計基礎，學會使用數種語言包含且不限於
JavaScript CSS HTML,Python, SQL, shell.

各種開發工具

熟悉資訊軟體業界開發工具使用 linux commandline,browser,web developer tool,IDE vscode,tmux

版本控制 git

品管程式碼，團隊合作到開放原始碼甚至現代開發的基石，版本控制。

開發與維運

現代軟體開發與維運開發流程工具，團隊程式開發，如何部署你的程式碼與服務。

網路概論

基本網路運作概論 tcp/ip,http,https,DNS,session,port

網頁前端

如何設計網頁，開發網頁程式前端服務，HTML,css,javascipt,sass,webpack, sketch,zepline.

網站後端

如何開發後端網站程式應用與維護，如何使用終端機，SQL,apache,nginx,Rest,ORM.

開放原始碼與商業

關於開放原始碼與現代商業資訊市場的些常識，lisence,cc,copyleft. 開放原始碼與商業市場的分分合合。

資安基礎

網路通訊安全基礎，對於現代軟體公司資訊安全的重要性，開發者應有的資訊安全意識，還有關於黑客的一些事。

project
