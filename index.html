現代資訊學院

在面對不確定的未來前，現在的你需要的創業、專案執行、自學與資訊能力

你不是一定要創業，但一定要擁有創業家的計畫思考。
你也不一定要撰寫程式碼，但一定要了解開發者的思考邏輯。

新創產業會變，程式碼規格會改，但是你對資訊背後所有原理的認知可以累積。

這裡是不在教室的未來大學，現代資訊學院，所有你需要的資源都已上線，可供你遠端連線進行學習。

什麼科系背景畢業不重要。

在本學院，藉由你有興趣的學習內容，學到教室裡學不到的資訊能力，打造在不確定的未來你所需要的自學能力以及解決問題的能力。

學院課程

course

現代網站開發課程

預計 3 月開始，為期半年到一年的線上課程，
以及進修線下活動。
學員可以自由選擇要如何參與課程及調整進度。

程式編寫

程式邏輯設計基礎，學會使用數種語言包含且不限於
JavaScript CSS HTML,Python, SQL, shell.

各種開發工具

熟悉資訊軟體業界開發工具使用 linux commandline,browser,web developer tool,IDE vscode,tmux

開發與維運

現代軟體開發與維運開發流程工具，團隊程式開發，如何部署你的程式碼與服務。

網路概論

基本網路運作概論 tcp/ip,http,https,DNS,session,port

網頁前端

網站後端

如何開發後端網站程式應用與維護，如何使用終端機，SQL,apache,nginx,Rest,ORM.

security
